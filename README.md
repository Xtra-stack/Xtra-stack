### 🧠 VIE — Virtual Industry Experience Platform

<img src="https://img.shields.io/badge/STATUS-LIVE-5DCAA5?style=flat-square&logo=vercel&logoColor=white"/> <img src="https://img.shields.io/badge/STACK-React%2018%20%7C%20Node.js%20%7C%20MongoDB%20%7C%20Docker-7F77DD?style=flat-square"/>

> *The closest thing to a real software job — without the office politics.*

```
  ┌─────────────────────────────────────────────────────────┐
  │  SDLC Simulation Flow                                    │
  │                                                          │
  │  [Junior Dev] → assign task                             │
  │       ↓                                                  │
  │  [Submit PR] → code review                              │
  │       ↓                                                  │
  │  [Senior Dev] → approve / request rework                │
  │       ↓                                                  │
  │  [Manager] → final sign-off ✓                           │
  └─────────────────────────────────────────────────────────┘
```

**What's inside:**
- Role-based system: `Junior → Senior → Manager` with structured approval gates
- In-browser coding with **Monaco Editor** (same engine as VS Code)
- JWT + RBAC on every API route
- Automated email notifications via Nodemailer on role transitions
- Dockerised, deployed on Vercel + Render behind **Nginx reverse proxy**
- Full **GitHub Actions CI/CD** pipeline
